# Abashkin_Spring

## Тренировочный проект для отработки навыков:
+ Sprig boot
+ Rest api
+ В качестве субд использовался Postgresql
+ Тесты выполнял через postman (json запросы)
+ В качестве сборщика использовал Gradle
___
## Отрабатывал:
+ связь сущностей
+ создание ошибок
+ создание сервисов
+ и работу с spring boot
